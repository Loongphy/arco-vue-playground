## supports

- vue3
- options and setup api
- typescript
- auto import arco and arco-icon
- code prettier
- defineModel
- reactivityTransform

> ‼️ arco-vue version support >= 2.28.0

tip: repl support prettier code use keyboard [mac] `Control + Shift + f`, [win] `Ctrl + Shift + f`
